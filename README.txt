Proyecto Final - Sitio Web Sencillo
Carpeta: Grupo_Perez_ProyectoFinal

Archivos:
- index.html  (Página 1: Vida Saludable)
- herramientas.html  (Página 2: Herramientas Creativas)
- redes.html  (Página 3: Fundamentos de Redes e IOT)
- salud.png
- gimp_genially_audacity.png
- relajacion.wav

Instrucciones rápidas:
1) Extrae la carpeta ZIP en tu equipo.
2) Abre index.html en un navegador (doble clic).
3) Para hospedar gratis: puedes usar GitHub Pages, Netlify Drop o Neocities.
   - GitHub Pages: sube la carpeta a un repositorio público y activa GitHub Pages en la rama main.
   - Netlify Drop: arrastra la carpeta a https://app.netlify.com/drop
   - Neocities: crea una cuenta y sube los archivos.

Nota: El video en index.html usa un enlace externo de ejemplo (sample-videos.com). Si necesitas que el video sea local, reemplaza la etiqueta <source> por un archivo .mp4 dentro de la carpeta.
